# Модель системи продажу та бронювання залізничних квитків

Проєкт виконано в межах лабораторних робіт 9-10 з дисципліни "Веб-програмування на стороні клієнта".

## 🚀 Технологічний стек
* React 18+ (Vite)
* React Router DOM v6
* State Management (useState, useContext)
* LocalStorage для збереження стану бронювань

## 📦 Функціонал
* Перегляд списку доступних потягів та рейсів
* Пошук та фільтрація рейсів
* Інтерактивна схема вагонів із відображенням вільних (зелений), обраних (синій) та заброньованих (червоний) місць
* Форма бронювання з валідацією введення персональних даних
